⚡ API — Multi-API Econômico Compacto
Recomendado: Econômico automático. O site tenta Gemini primeiro; se bater limite, tenta OpenRouter, Groq e Mistral.
👁️ Google Gemini — principal para visão
Use para prints, imagens de referência, análise visual, pacote completo e cenas com imagem. Pode bater limite gratuito.
🔁 OpenRouter — fallback gratuito para texto
Use quando o Gemini travar. Bom para roteiro textual, Narrador IA, prompts, SRT, CapCut e storyboards textuais.
⚡ Groq — texto rápido
Use para narrador, resumo, legendas, SRT, limpeza de texto e prompts rápidos. Não é o foco para visão.
🧠 Mistral — revisão e acabamento
Use para revisar, organizar, limpar texto, melhorar prompts e gerar narração. Bom como fallback extra.
📌 Como usar sem se perder
1. Deixe o modo em Econômico automático.
2. Clique em cada aba: Gemini, OpenRouter, Groq e Mistral.
3. Cole cada chave no campo correto.
4. Clique em Salvar chaves.
5. Clique em Testar modo.
6. Volte no Turbo e gere o pacote.
Para uso local/pessoal, salvar no navegador ajuda. Para publicar online, coloque as chaves em backend/proxy, não no HTML público.
