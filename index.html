Produção criativa em nodes para vídeo, gráfica e automação com IA.

A plataforma agora se organiza por motores independentes. O Studio Remotion Lab é o laboratório do criador, Video Remotion é o produto audiovisual final, Print é o produto gráfico/pré-impressão e API/Config é o motor técnico global.

Studio Remotion Lab

laboratório do criador

Área interna para testes, Fruits, presets manuais, microagentes, ideias novas, versões Titan/Colossal e experimentos de interface.

Setup limpo → Storyboard → Cena Motion 2D → SRT/Som → CapCut Book Exportar → Testes → Versões
🎬

Video Remotion

produto audiovisual final

Motor principal para criar vídeo com fluxo limpo: briefing, roteiro, storyboard, motion, Remotion JSON, SRT, som, CapCut Book e pacote final.

Briefing Mestre → Roteiro/Cena Storyboard → Motion 2D / Vídeo IA Remotion JSON → SRT/Som → Pacote Final
🖨️

Print

gráfica e pré-impressão

Motor gráfico para atendimento, produto, referências, orçamento, direção de arte, copy, prompt IA, pré-impressão, produção e entrega.

Atendimento → Produto Gráfico → Orçamento Direção de Arte → Copy → Prompt IA Pré-impressão → Produção → Pacote Final
motor técnico global

API / Config

Esta área não é produto criativo. Ela centraliza segurança, autenticação, backend e provedores de IA para todos os motores da plataforma.

Auth: ADMIN_USER, ADMIN_PASS e SESSION_SECRET protegidos no backend.
IA: Claude, Gemini, Groq e OpenRouter podem ser ligados por rotas internas.
Regra: nenhuma chave deve ficar exposta em HTML público.
Fluxo: Video Remotion, Print e Lab chamam a API, mas não mostram a chave ao usuário.
