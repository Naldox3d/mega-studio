🍓 Fruits Adventures Studio Standalone

HTML único com API Config temporária. Cole Gemini, Groq ou OpenRouter, gere história, personagens, model sheets, storyboard e pacote final.
IA: configure a API

Entrada do projeto

Use isso como ferramenta de trabalho pessoal. As chaves da API ficam temporárias na memória da página e somem ao atualizar.
Para vídeo, envie prints/frames principais. Gemini aceita imagem melhor; OpenRouter depende do modelo; Groq aqui é recomendado para texto.

Resultado rápido

Configure a API, cole a ideia e clique em Gerar com IA.

Gemini

Melhor para imagem/prints e análise visual. Chave temporária: some se atualizar a página.

OpenRouter

Bom para modelos gratuitos/alternativos e pacote grande. Alguns modelos aceitam imagem.

Groq

Rápido para texto, storyboard, narração e limpeza. Use sem imagem nesta versão.

Modo de IA

Nada é salvo no navegador. Se atualizar a página, precisa colar a chave de novo.
