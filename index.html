Fruits Adventures Studio

Motor de criação com provedores de IA: Gemini, Groq e OpenRouter. Ele transforma vídeo, imagem, ideia ou universo em história, personagens, model sheets, storyboard e prompts finais.

🎬 Entrada do projeto

Para GitHub Pages, use Rascunho ou conecte um backend externo. Nunca coloque chave de API direto no HTML público.

Imagem funciona melhor. Para vídeo, o backend envia o arquivo quando o provedor aceitar; se não aceitar, use frames/imagens.

Status: se estiver no GitHub Pages, está em frontend. Para usar chave API, rode o backend local ou em Render/Railway/VPS.
