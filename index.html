📚 Guia rápido — APIs gratuitas

🏆 Gemini (recomendado) 1. Acesse aistudio.google.com
2. Crie uma chave gratuita
3. Cole acima em "Gemini API Key"
4. Tier gratuito: 1500 req/dia
5. Suporta imagens e video frames
⚡ Groq (mais rápido) 1. Acesse console.groq.com
2. Crie conta e gere chave
3. Cole em "Groq API Key"
4. 14.400 tokens/minuto grátis
5. Melhor para texto e storyboard
🔀 OpenRouter (flexível) 1. Acesse openrouter.ai
2. Crie conta gratuita
3. Cole em "OpenRouter API Key"
4. Modelos gratuitos disponíveis
5. Ótimo como backup
